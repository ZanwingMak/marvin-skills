| Country | Code | Language |
|---------|------|----------|
| United States | US | English |
| Hong Kong | HK | Chinese |
| Taiwan | TW | Chinese |

| South Korea | KR | Korean |

| Australia | AU | English |
| Canada | CA | English |
| Singapore | SG | English |
| India |English |
| China IN | Hindi/ | CN | Chinese |
| Thailand | TH | Thai |
| Malaysia | MY | Malay |

## URL Parameters

- `geo={CODE}` - Region code
- `sort=search-volume` - Sort by search volume
- `hours={4|24|48|168}` - Time range in hours (4h, 24h, 48h, 7days)

## Example URLs

```
US 4h:   https://trends.google.com/trending?geo=US&sort=search-volume&hours=4
HK 24h:  https://trends.google.com/trending?geo=HK&sort=search-volume&hours=24
TW 48h:  https://trends.google.com/trending?geo=TW&sort=search-volume&hours=48
JP 7d:   https://trends.google.com/trending?geo=JP&sort=search-volume&hours=168
KR 24h:  https://trends.google.com/trending?geo=KR&sort=search-volume&hours=24
```
